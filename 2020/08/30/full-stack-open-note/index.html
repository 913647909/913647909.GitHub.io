JSX

    Hook

  • Hook的规则

    • 不能从循环、条件表达式或任何不是定义组件的函数的地方调用 useState

  • state hook

    • 导入 useState 函数
      • import React, { useState } from 'react'
    • 定义组件的函数体
      • const [ counter, setCounter ] = useState(0)
    • 每次 setCounter 修改状态时，它都会导致组件重新渲染

  • effect hooks

    • 从服务器获取数据时使用的正确工具

    • 在函数组件中执行副作用

    • 数据获取、设置订阅和手动更改 React 组件中的 DOM
